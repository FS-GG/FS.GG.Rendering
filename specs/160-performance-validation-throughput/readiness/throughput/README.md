# Feature 160 Focused Throughput Evidence

Entry point: `summary.md`.

This directory contains the bounded focused performance lane for Feature 160:

- `summary.md` and `summary.json` publish the run-level status.
- `iterations/` contains one markdown record per focused iteration.
- `raw/` contains CSV and JSON sample evidence for the required Feature 158 timing scenarios.
- `excluded/` groups evidence that contributes zero accepted throughput.
- `unsupported/` records unsupported-host validation with zero accepted same-profile performance artifacts.

Focused throughput collection does not run broad solution validation.
