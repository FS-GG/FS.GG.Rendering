# Ant form template — recipe (groundwork)

> Ant is adopted as a design language only — no React/DOM/HTML/CSS dependency.

**Status: groundwork.** Target shape for D3/G3, not shipped behavior. A form page stacks labelled
fields (`text-box`, `check-box`) with a primary submit `button`, all on `Seed.controlHeight` (32)
and the 8-unit grid. Built from existing input/selection control families.
