# Ant exception template — recipe (groundwork)

> Ant is adopted as a design language only — no React/DOM/HTML/CSS dependency.

**Status: groundwork.** Target shape for D3/G3, not shipped behavior. An exception page handles
error states (403/404/500): a large status `icon`, a `text-block` explanation, and a recovery
`button`, with generous `Space.xl` spacing. Built from existing display/input control families.
