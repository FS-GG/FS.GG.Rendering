# Ant result template — recipe (groundwork)

> Ant is adopted as a design language only — no React/DOM/HTML/CSS dependency.

**Status: groundwork.** Target shape for D3/G3, not shipped behavior. A result page reports the
outcome of an operation: a status `icon`, a `text-block` headline/description, and a primary
`button` call-to-action, centered with `Space.lg` spacing. Built from existing display/input
control families.
