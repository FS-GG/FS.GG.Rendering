# Ant workbench template — recipe (groundwork)

> Ant is adopted as a design language only — no React/DOM/HTML/CSS dependency.

**Status: groundwork.** This is the target shape for Workstream D3 (kits) / G3 (Ant showcase), not
shipped behavior. A workbench is a dense, multi-pane operations screen: a top `toolbar`, a primary
`data-grid` working area, and side `panel`s for filters/detail — composed only of existing control
families on the 8-unit grid (`Space.lg` gutters).
